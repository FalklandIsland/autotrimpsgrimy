AutoTrimps Grimy Perk Optimizer Patch (Steam mods.js friendly)

What this adds
- New setting: "Use Grimy Perk Optimizer" (Portal / Universe 1).
- When enabled, Auto Allocate Perks will run the optimizer logic copied from Grimy's perk calculator (grimy.github.io src/perks.ts).
- If the Grimy optimizer throws an error for any reason, AutoTrimps automatically falls back to the built-in Perky optimizer and logs an error to the console.

Also changed
- AutoTrimps2.js now auto-detects its basepath from the URL it was loaded from. This makes forks work without hardcoding URLs.
  (If auto-detection fails, it falls back to SadAugust's GitHub Pages.)

Files modified
- AutoTrimps2.js
- SettingsGUI.js
- mods/perky.js

Steam usage
1) Host this repo on GitHub Pages (or any static host).
2) Update Trimps Steam mods.js to load your hosted AutoTrimps2.js.
3) In-game: Settings -> Portal -> enable "Auto Allocate Perks" and "Use Grimy Perk Optimizer".
